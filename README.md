# Red Wine Quality Analysis

## Overview
This project performs exploratory data analysis (EDA) on the Red Wine dataset to understand the factors affecting wine quality.

## Dataset
The dataset contains physicochemical features such as acidity, pH, alcohol content, and a quality score.

## Key Steps
- Data cleaning (removing duplicates)
- Statistical summary
- Correlation analysis
- Visualization (histograms, boxplots, scatter plots)
- Feature relationship analysis

## Key Insights
- Alcohol has a strong positive correlation with quality
- Volatile acidity negatively affects quality
- The dataset is imbalanced, with most wines having average quality

## Tools Used
- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
